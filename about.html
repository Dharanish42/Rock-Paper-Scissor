About the Game

Rock Paper Scissors is a classic hand game that is believed to have originated in China as far back as the Han Dynasty (206 BCE – 220 CE). The game spread to Japan where it became a popular game called “Jan-Ken”. The modern version we know today has been popular worldwide for generations.

About This Webpage

This webpage version of the game was created by Dharanish S, a Fullstack Developer with a passion for creating engaging and interactive web experiences. This project is a simple yet entertaining take on the traditional game.

The game uses HTML, CSS, and JavaScript to provide a seamless experience, allowing users to play against a computer and enjoy the fun dynamics of the game in a digital format.
